🚀 Mini Project Showcase – Gym Website (FitFlux)

I recently built a responsive gym website as a mini front-end project using HTML, CSS, and JavaScript. The goal of this project was to strengthen my core web development skills by designing a clean user interface and creating a structured multi-section layout.

🔹 Key Features:
• Modern landing section with motivational hero banner
• Trainers and workout categories with organized card layouts
• Weekly classes and equipment sections
• Contact form UI
• Responsive design with simple and consistent styling

Through this project, I focused on improving layout design, CSS styling, component structure, and basic JavaScript interactions. It helped me understand how real-world fitness or business websites are structured from a front-end perspective.

This is a small but important step in my journey as a front-end developer, and I’m continuously working on improving my projects and learning new concepts.
